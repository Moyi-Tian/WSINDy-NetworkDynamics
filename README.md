# WSINDy-NetworkDynamics

This repository contains the code used for the paper:

"Learning Effective Models from Network Dynamics Data with Multiple Initial Conditions Using Weak Form SINDy"



The code implements numerical experiments for learning effective ordinary differential equation models from synthetic data generated by fully-mixed and stochastic network dynamical systems. The main inference method is Weak Form Sparse Identification of Nonlinear Dynamics, WSINDy, applied to data generated from multiple initial conditions.



## Dependency

This repository uses the WSINDy implementation from:

https://github.com/dm973/wsindy_obj_base

The experiments were run using the main branch of `wsindy_obj_base` as downloaded on June 25, 2025.

Before running the scripts, replace every instance of

```matlab
provide_path_to_WSINDy
```

with the local path to the directory containing the WSINDy source code.



## Repository Structure

### `FullyMixed_E0`

This folder contains experiments for the fully-mixed model with varying initial conditions in the engagement variable \(E\). These scripts reproduce the fully-mixed model results reported in the paper.

### `Erdos-Renyi`

This folder contains experiments for stochastic dynamics on Erdős-Rényi networks. These scripts reproduce the network-based stochastic model results reported in the paper.

### `FullyMixed_P0`

This folder contains additional experiments for the fully-mixed model with varying initial conditions in the protest variable \(P\). These experiments generate data for extended results discussed in the paper, but not explicitly reported as figures.



## Running the Experiments

Each main experiment folder contains scripts for data generation, WSINDy inference, and figure generation.

### 1. Generate synthetic data

In each experiment folder, first run the corresponding driver script in the `Generate_Data` subfolder. These scripts have filenames beginning with

```matlab
driver
```

For example:

```matlab
./FullyMixed_E0/Generate_Data/driver*.m
```

This step generates the synthetic trajectory data used for model discovery.

### 2. Run WSINDy

After generating the data, run the WSINDy learning scripts. These scripts have filenames beginning with

```matlab
WSINDyLearn
```

These scripts apply WSINDy to the generated data and save the learned model results.

### 3. Generate figures

Scripts with filenames containing

```matlab
_Fig
```

generate the corresponding figures in the paper.

Some figure-generation scripts require additional preprocessing before plotting. When this is needed, the required preprocessing step is noted at the top of the corresponding plotting script.



## Notes

All synthetic data used in the experiments can be generated locally by running the provided data-generation scripts. Each experiment folder is organized so that it can be run independently after the WSINDy dependency path has been set correctly.
